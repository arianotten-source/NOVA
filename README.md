# N.O.V.A. — Neural Observation & Voice Assistant

Persoonlijke AI-assistent met React, Electron en lokale JSON-opslag.

## Quick start

```bash
npm install
npm run dev          # Browser + Sensor API
npm run electron:dev # Desktop app
npm run build        # Full build (web + Electron)
npm run build:web    # Web-only build (Vercel)
```

## Deploy

Zie [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) voor GitHub en Vercel instructies.

## Tech stack

- React 19 · TypeScript · Vite · TailwindCSS
- Electron (desktop)
- Vercel (web PWA)
